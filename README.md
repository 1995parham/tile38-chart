## Introduction

Helm chart for [tile38](https://github.com/tidwall/tile38).
You can use it as follows:

```bash
helm repo add tile38 https://1995parham.me/tile38-chart

helm show chart tile38/tile38
```

Or add it into your chart as dependency.
